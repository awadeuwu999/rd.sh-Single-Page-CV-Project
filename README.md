# About
This is a single-page CV project made in HTML to show my skills and education as a Mechatronics engineer.

# How to view 
Download this repository on your PC, then go to the place where you downloaded it, finally, double click on the "CV.html" file and it'll be open in your default browser (Chrome, Firefox, Edge, etc). If it's necessary, click right button on the "CV.html" file and select the 'open with' option so you could open the file with the browser that you want.
